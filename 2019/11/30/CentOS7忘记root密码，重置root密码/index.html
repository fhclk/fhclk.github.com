CentOS7忘记root密码，重置root密码

发表于 2019-11-30

  1. 重启系统，在开机过程中，出现以下界面时按e键，进入编辑界面。

    按e键之后进入如下界面：

  2. 按方向键下键↓，找到设置语言的地方，如LANG=en_US.UTF-8，在后面追加rw single init=/bin/bash,然后按ctrl+x重启系统

  3. 进入bash界面后，可以输入passwd命令重新设置root密码
